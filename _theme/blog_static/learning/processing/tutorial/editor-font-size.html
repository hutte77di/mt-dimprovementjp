エディタの文字サイズを変更する

2012年5月 6日 / 2020年4月18日更新

エディタの文字サイズはデフォルトで10ptと小さく読みづらい。文字サイズを大きくしたい場合は、環境設定から変更できる。

見やすくなった。筆者は14ptに設定した。

開発環境の右下をドラッグすることで開発環境のサイズを調整できる。入力されたコードが読みやすくなるように、サイズを調整して制作の効率を上げよう。
